Build a full responsive web app called “OTV Card Maker” or “Premium Bangla News Card Generator”.

Goal:
Create a professional template-based social media card generator for Bangla news graphics, quote cards, bulletin cards, editorial cards, portrait cards, and breaking news cards. The app should feel polished, modern, premium, and production-ready.

Tech stack:
- React
- TypeScript
- Vite
- Tailwind CSS
- HTML5 Canvas for rendering
- No backend required
- Everything should run client-side in browser
- Organize code cleanly into reusable components and utilities

Main app layout:
- Left sidebar or left control panel for editing options
- Center or right side large live preview area
- Top bar with branding and template navigation
- Responsive layout for desktop first, but mobile also usable

Core features:
1. Create square cards in 1080x1080
2. Support multiple card templates
3. Live preview updates instantly
4. Upload main image
5. Upload logo
6. Add headline
7. Add subheadline
8. Add bullet text
9. Add quote text
10. Add source / courtesy / via text
11. Add speaker name and designation
12. Highlight selected text segments
13. Toggle watermark
14. Toggle background grid lines
15. Toggle sandy grain / paper texture
16. Choose accent colors
17. Download final output as PNG
18. Reset template
19. Switch between different preset templates
